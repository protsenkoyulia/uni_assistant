# uniHelper — чат-бот для адаптации китайских студентов в СПбГЭТУ «ЛЭТИ»

**uniHelper** — это интеллектуальный Telegram-бот, разработанный для помощи китайским студентам, обучающимся в Санкт-Петербургском государственном электротехническом университете «ЛЭТИ» им. В.И. Ульянова (Ленина). Бот предоставляет актуальную информацию о университете, жизни в Санкт-Петербурге, учебном процессе, общежитиях, документах и повседневных вопросах, а также выполняет контекстный перевод и поддерживает диалог на трёх языках (русский, китайский, английский).

### Основные возможности
- Ответы на вопросы об университете ЛЭТИ на основе базы знаний
- Контекстный перевод текстов (русский ↔ китайский ↔ английский) с учётом истории диалога
- Сохранение и использование истории переписки для связных ответов
- Многоязычный интерфейс и клавиатура
- Поддержка команд и кнопок: /start, /help, /language, /clear, /translate
- Постоянное обновление базы знаний через добавление новых документов

### Технологии

| Компонент                  | Технология / Библиотека                  | Назначение                                      |
|----------------------------|------------------------------------------|-------------------------------------------------|
| Telegram-интерфейс         | aiogram                              | Асинхронный фреймворк для Telegram Bot API      |
| Языковая модель + RAG      | GigaChat (Sber) + LangChain              | Генерация ответов и контекстный поиск           |
| Векторная база знаний      | Weaviate                                 | Хранение и семантический поиск документов       |
| Загрузка и чанкинг         | LangChain (Docx2txtLoader + RecursiveCharacterTextSplitter) | Парсинг .docx и разбиение на чанки              |
| Кэш контекста и языка      | Redis                                    | Хранение истории диалога и выбранного языка     |
| Эмбеддинги                 | GigaChatEmbeddings                       | Векторизация текстов для Weaviate               |
| Управление зависимостями   | python-dotenv                            | Работа с .env-файлом                            |

### Установка и запуск

#### Требования
- Python 3.9–3.12
- Redis (локальный или облачный)
- Weaviate (локальный Docker-контейнер или облачный)
- Учётная запись GigaChat API (токен)
- Telegram-бот (токен от @BotFather)

#### Шаг 1. Клонируйте репозиторий
```bash
git clone https://github.com/ваш-username/unihelper.git
cd unihelper
```

#### Шаг 2. Создайте и активируйте виртуальное окружение
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux / macOS
source venv/bin/activate
```

#### Шаг 3. Установите зависимости
```bash
pip install -r requirements.txt
```

#### Шаг 4. Настройте переменные окружения
Создайте файл .env в корне проекта:
```bash

# GigaChat
AUTH_KEY_GIGACHAT=
MODEL_GIGACHAT=GigaChat-Pro

# TG
TG_TOKEN=

# VK
VK_TOKEN=
VK_GROUP_ID=

# PostgreSQL
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DB=uni_assistant_ai
POSTGRES_USER=postgres
POSTGRES_PASSWORD=

# Redis
REDIS_HOST=localhost
REDIS_PORT=6379
 
# RAG настройки
VECTOR_DB_PATH=./vector_db
```

#### Шаг 5. Запустите Docker
Bash
```bash
docker run -d -p
```

#### Шаг 6. Запустите бота
Bash
```bash
python main.py
```
